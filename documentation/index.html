System Requirements

  • Node.js v12.22.0 or later
  • macOS, Windows (including WSL), and Linux are supported

Application Requirements

  • Cloudinary Account
    • What is Cloudinary?
      • Cloudinary provides cloud-based image and video management services. It enables users to upload, store, manage, manipulate, and deliver photos and videos for websites and apps.
      • And it’s free; you can register from here.
  • Stripe Developer Account.
    • And create a publishable key & Secret Key.
  • MySQL
    • MySQL is an open-source relational database management system.

Quick Installation Guide

  • Create Database & Import included SQL file that comes with the Edmy zip file.
  • Open the terminal window into the edmy-react-next folder.
  • Run npm install.
  • Run npm run dev.
  • Open the browser and copy paste the link- http://localhost:3000 in the browser address bar.
  • The website will load, and the process is done!

Setup Development Environment

Application Install Preparation

  • Install Node.js
    • NodeJS works as a node package manager behind the scene. It is recommended to download and install the latest version of Node.js.
  • Install WAMP for MySQL/PHPMyAdmin.
  • Create Cloudinary Account And Follow the below screenshots.




1. https://api.cloudinary.com/v1_1/cloud-name/image/upload
2. https://api.cloudinary.com/v1_1/cloud-name/video/upload
  • Create Stripe Account

    • Now from the application root directory, open the next.config.js file using your code editor and update like below-

    CLOUDINARY_URL: "https://api.cloudinary.com......", CLOUDINARY_VIDEO_URL: "https://api.cloudinary.com......", STRIPE_SECRET_KEY: "sk_test_3D..........", STRIPE_PUBLISHABLE_KEY: "pk_test_Zy........"

Finish Setup & Run the App

Now, open the terminal and cd to your project root folder. After that, you need to run the following commands-

npm install
npm run dev
